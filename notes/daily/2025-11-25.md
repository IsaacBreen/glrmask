### Paper Details

- **Title:** "Efficient Grammar-Constrained Decoding via Precomputed Deterministic Weighted Automata"
- **Location:** `paper/main.tex` (761 lines)
- **Compiled:** `paper/main.pdf` (14 pages, 480KB)

### Paper Structure

1. **Abstract** — Problem statement, key contributions, results preview
2. **Introduction** — Motivation, alignment problem, existing approaches, our contributions
3. **Background** 
   - Context-free grammars and GLR parsing
   - Subword tokenization (BPE)
   - The constraint computation problem
4. **Method**
   - System overview with pipeline diagram
   - Compilation pipeline (grammar analysis, tokenizer, product automaton)
   - Deterministic weighted automata with bitvector semiring
   - Optimizations: state equivalence, sparse bitvectors, template caching
5. **Theoretical Analysis**
   - Compilation complexity
   - Runtime complexity (O(1) per token)
   - Soundness and completeness proofs
   - Space complexity
6. **Experiments**
   - Setup: JSON/JavaScript/SQL grammars, GPT-2 vocab, baselines
   - Per-token latency: 30-100× speedup vs Outlines
   - Compilation time: 2-45 seconds depending on grammar
   - State equivalence: 52-62% vocabulary reduction
   - End-to-end: <1% constraint overhead
7. **Related Work** — GCD, Outlines, XGrammar, Guidance, LMQL, weighted automata
8. **Discussion** — Limitations, future work
9. **Conclusion**
10. **Appendix** — Algorithm pseudocode, grammar specs, implementation details
